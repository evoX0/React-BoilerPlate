# React-BoilerPlate
A pure boiler plate for React with:
 - configured Webpack
 - Babel
 - Eslint
 - Prettier
 - SASS
 - CSS
 
 # With Redux
 user  `with-redux` branch

To get started you just need to run the following:
- npm install or yarn
- npm start to run the project
- npm run build for production build
